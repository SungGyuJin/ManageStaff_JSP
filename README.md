# ManageStaff_JSP

### 프로젝트 제작 배경

> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;학원에서 JSP Model 1을 공부중 가장 처음 배운 개념입니다. 기타 자질구레한 기능등을 다 제외하고 순수 등록, 조회, 수정, 삭제의 개념만 이용한 게시판개념의 프로젝트입니다.

### 설명
